# AWS Sample - Lambda with EventBridge Scheduler

## Introduction

This repository contains sample code that invokes a Lambda function on a schedule using EventBridge Scheduler.

## Usage

### Build project

```shell
mvn clean install
```

### Deploy Lambda with scheduler

```shell
terraform apply
```
